Política de Privacidade — Ah Lembrei

Última atualização: Maio de 2026

O aplicativo Ah Lembrei foi desenvolvido com o objetivo de ajudar usuários a armazenar informações importantes de forma prática, segura e organizada.

1. Coleta de Informações

O aplicativo pode armazenar informações fornecidas diretamente pelo usuário, incluindo:

  • Senhas;
  • Documentos;
  • Contatos;
  • Anotações pessoais;
  • Outras informações inseridas manualmente.

Nenhuma informação é coletada sem o consentimento do usuário.

2. Armazenamento dos Dados

Os dados são armazenados localmente no dispositivo do usuário, salvo quando houver integração opcional com serviços externos autorizados pelo próprio usuário.

3. Compartilhamento de Dados

O aplicativo não vende, compartilha ou distribui informações pessoais para terceiros.

Os dados permanecem sob controle exclusivo do usuário.

4. Serviços de Terceiros

O aplicativo pode utilizar serviços de terceiros, como:

  • Google AdMob;
  • Firebase Analytics;
  • Google Play Services.

Esses serviços podem coletar informações não pessoais para melhorar a experiência do usuário e exibir anúncios relevantes.

5. Segurança

Adotamos medidas razoáveis de segurança para proteger as informações armazenadas no aplicativo.

Entretanto, nenhum método de armazenamento eletrônico é totalmente seguro, e não podemos garantir segurança absoluta.

6. Direitos do Usuário

O usuário pode:

  • Excluir informações armazenadas;
  • Desinstalar o aplicativo a qualquer momento;
  • Solicitar esclarecimentos sobre esta política.

7. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada periodicamente.

Recomendamos que o usuário revise esta página regularmente para acompanhar possíveis alterações.
